Extract Data from Invoices

This API allows you to extract structured data from invoice files (PDF, PNG, JPEG) using Optical Character Recognition (OCR) technology.

Upload an Invoice API Documentation

Features

  • Upload and process invoice files
  • Extract key information like invoice number, date, and amount
  • Return structured data in JSON format
  • Authentication to protect your data
  • Simple web interface or API access

Using the API

You can use the API directly via HTTP requests:

curl -X POST "http://localhost:8000/extract/" \
  -H "Authorization: Basic YWRtaW46cGFzc3dvcmQ=" \
  -F "file=@/path/to/invoice.pdf"

Or use the web interface for an easier experience.
